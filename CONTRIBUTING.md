\# Contributing to IssueFlow Contracts



Thank you for contributing! 🎉



\## Prerequisites

\- Rust installed

\- Stellar CLI installed

\- Soroban target: `rustup target add wasm32v1-none`



\## Setup

1\. Fork the repository

2\. Clone your fork:

```bash

&#x20;  git clone https://github.com/YOUR\_USERNAME/issueflow-contracts.git

&#x20;  cd issueflow-contracts

```

3\. Build the contracts:

```bash

&#x20;  cargo build --target wasm32v1-none --release

```

4\. Run tests:

```bash

&#x20;  cargo test

```



\## How to Contribute

1\. Check open issues tagged `good first issue`

2\. Comment on the issue to claim it

3\. Create a branch: `git checkout -b feature/your-feature`

4\. Make your changes

5\. Push and open a Pull Request



\## Commit Style

\- `feat:` new feature

\- `fix:` bug fix

\- `docs:` documentation

\- `chore:` maintenance



\## Response Time

Maintainers will review PRs within \*\*48 hours\*\*.
